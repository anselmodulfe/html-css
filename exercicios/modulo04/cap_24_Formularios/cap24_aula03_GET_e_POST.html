Anotações da aula

Em HTML os métodos GET e POST podem ser escritos em letras maiúsculas ou minúsculas

O método default é o GET

O método GET tem limite de envio: 3000 bytes e não envia arquivos

O método POST é ideal para dados sensíveis, envios com mais de 3000 bytes e arquivos

O método GET é mais rápido porem a diferença não é sensível

Problemas nos formularios desse aula.

  • Os dados são enviados mesmo se náo existirem, caixa em branco
